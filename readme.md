Step to run application:
Step 1: Create the copy of the project.
Step 2: Open command prompt and change your current path
to folder where you can find 'app.py' file.
Step 3: Check for any pip updates, if yes update them.
Step 4: Open annaconda prompt and install streamlit by bleow command-
pip install streamlit.
Step 5: Use command below to install required dependencies-
python -m pip install -r requirements.txt
Step 6: Run application by command-
streamlit run app.py
You will get url copy it and paste in browser.
Step 7: You have sample_data folder where you can get images to test.

NOTE: If youy are using your own pictures for sample data make sure they are in .png foramt.
